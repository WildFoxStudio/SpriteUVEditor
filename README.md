# SpriteUVEditor
Spritesheet uv editor, animation preview and metadata export.

## Features
- Spritesheet uv editor
- Animation preview
- Metadata export

## Releases
Download the latest release from [releases](https://github.com/VolpinGames/SpriteUVEditor/releases).

## Installation
Use cmake to build the project.

## Requirements
 - CMake at least version 3.22
 - C++17 compiler

Feel free to contribute to the project by creating a pull request.
